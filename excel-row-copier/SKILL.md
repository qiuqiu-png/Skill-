# 表格复制行数

批量复制Excel/WPS表格的指定行数据。

## 功能说明

这个skill可以读取Excel文件的指定列和行数，然后将每行数据按指定次数复制，生成新的Excel文件。

## 使用方法

1. 准备好需要处理的Excel文件
2. 运行脚本并按提示输入以下参数：
   - 输入文件路径
   - 输出文件路径
   - 要读取的列（如：A:A 或 A:C）
   - 要读取的行数（从第一行开始）
   - 每行复制的次数

## 示例

```
输入文件: /Users/nanyu/Desktop/复制专用.xlsx
输出文件: /Users/nanyu/Desktop/输出.xlsx
读取列: A:A
读取行数: 22
每行复制次数: 128
```

## 依赖项

- pandas
- openpyxl

## 注意事项

- 确保输入文件存在且可访问
- 输出目录会自动创建
- 复制次数建议根据实际需求设置，避免生成过大的文件
