# 新店批发毛利明细分析

自动分析2025年新店的批发额和毛利情况,按每个新店单独统计,包含开业时间和按件/按克批发额明细。

## 核心功能

1. **新店识别**: 根据组织匹配表自动识别2025年新开店
2. **剔除闭店**: 自动过滤已关闭的门店数据
3. **M结尾过滤**: 只分析组织名称以M结尾的门店
4. **开业时间**: 从组织匹配表读取每个新店的开业日期
5. **按件/按克区分**: 根据计量单位('件'或'克')分别统计批发额
6. **计算指标**:
   - 销售金额
   - 退货金额
   - 批发额(含税) = 销售金额 - 退货金额
   - 按件批发额(含税) = 计量单位='件'的批发额
   - 按克批发额(含税) = 计量单位='克'的批发额
   - 毛利额(未税) = (销售金额 - 暂估成本)/1.13 - (退货金额 - 暂估成本)/1.13
   - 毛利率(%) = 毛利额 / 批发额 × 100%

## 使用方法

### 准备数据文件

需要准备以下3个Excel文件:

1. **销售结算明细表**: 包含"组织"、"计量单位"、"销售金额"、"暂估成本"等列
2. **退货结算明细表**: 包含"组织"、"计量单位"、"退回金额"、"暂估成本"等列
3. **组织匹配表**: 包含"加盟门店"、"开业日期"、"2025年新店"、"已闭店"等列

### 运行分析

```bash
python3 ~/.claude/skills/new-store-detail-analysis/analyze_new_stores.py <销售文件> <退货文件> <组织匹配表> [输出文件]
```

示例:

```bash
# 默认输出文件名
python3 ~/.claude/skills/new-store-detail-analysis/analyze_new_stores.py 销售.xlsx 退货.xlsx 组织匹配.xlsx

# 指定输出文件名
python3 ~/.claude/skills/new-store-detail-analysis/analyze_new_stores.py 销售.xlsx 退货.xlsx 组织匹配.xlsx 新店明细.xlsx
```

### 输出说明

生成的Excel文件包含以下列:

- **新店组织名称**: 新店的完整名称
- **开业时间**: 新店的开业日期(来自组织匹配表)
- **销售金额**: 该店的总销售金额
- **退货金额**: 该店的总退货金额
- **批发额(含税)**: 销售金额 - 退货金额
- **按件批发额(含税)**: 计量单位='件'的批发额
- **按克批发额(含税)**: 计量单位='克'的批发额
- **毛利额(未税)**: 未税毛利额
- **毛利率(%)**: 毛利率百分比

最后一行为"合计"行,汇总所有新店的数据。

## 输出格式

- 专业Excel格式,包含:
  - 蓝色表头背景 + 白色字体
  - 数字千分位格式
  - 百分比格式
  - 所有单元格边框
  - 合计行加粗
  - 底部注释说明

## 工作流程

1. **读取组织匹配表**: 识别新店、已闭店和开业日期
2. **读取销售和退货数据**: 加载并处理数据
3. **M结尾过滤**: 只保留组织名称以M结尾的门店
4. **剔除已闭店**: 过滤掉已关闭门店的所有数据
5. **只保留新店**: 仅分析2025年新店数据
6. **按店统计**: 为每个新店计算各项指标
7. **区分按件/按克**: 根据计量单位分别统计批发额
8. **生成报表**: 输出包含合计行的完整报表

## 注意事项

- 组织名称会自动处理,去除末尾的*和M字符
- 只分析组织名称以M结尾的门店,其他门店自动排除
- 已闭店的数据会被自动剔除
- 输出为净批发额、净毛利额数据
- 按件/按克批发额之和等于总批发额
